# Events & Broadcasting

> Create event pages for funerals, memorial services, celebrations of life, and repasts. Share with family and friends, and track RSVPs.

## What You'll Need
- An active FinalWishes account with at least one estate
- Principal, admin, or executor role on the estate

## Step by Step

### Creating an Event

1. Navigate to your estate and click **Events** under the Letters section in the sidebar
2. Click the **Create Event** button
3. Fill in the event details:
   - **Event Type** -- Choose from: Funeral Service, Memorial Service, Celebration of Life, Repast/Reception, Graveside Service, or Other
   - **Title** -- A name for the event (e.g., "Memorial Service for James")
   - **Date and Time** -- When the event starts (and optionally when it ends)
   - **Location** -- Venue name
   - **Address** -- Full street address for directions
   - **Description** -- Details about the event, parking, or special instructions
   - **Dress Code** -- Optional (e.g., "Business casual" or "Wear something colorful")
   - **Notes** -- Any additional information for attendees
4. Toggle **RSVP Tracking** on if you want to collect responses
5. Click **Save**

### Managing Events

- **Edit** -- Click the pencil icon on any event to update details
- **Cancel** -- Mark an event as cancelled (attendees will see the updated status)
- **Delete** -- Permanently remove an event (confirmation required)

Events are displayed with status badges:
| Status | Color | Meaning |
|--------|-------|---------|
| **Upcoming** | Blue | Event has not yet occurred |
| **Completed** | Green | Event date has passed |
| **Cancelled** | Red | Event was cancelled |

### RSVP Tracking

When RSVP is enabled on an event:
1. Click the **RSVPs** button to open the RSVP panel
2. See a summary of responses: Attending, Declined, Maybe
3. Share the event link with family -- they can respond directly
4. The RSVP count updates on the event card

### Sharing Events

Click the **Copy Link** button on any event to copy a shareable link to your clipboard. This link can be sent via email, text, or social media. Recipients do not need a FinalWishes account to view the event details.

Events can also be connected to **Public Memorials** -- if your estate has a memorial page with a QR code, the event details can appear there for visitors.

## Event Types Explained

| Type | When to Use |
|------|-------------|
| **Funeral Service** | Traditional funeral, often at a funeral home or place of worship |
| **Memorial Service** | Service held without the body present, often after cremation |
| **Celebration of Life** | Less formal gathering focused on celebrating the person's life |
| **Repast / Reception** | Gathering after the funeral for food and fellowship |
| **Graveside Service** | Service held at the burial site |
| **Other** | Any other type of gathering (reading of the will, family meeting, etc.) |

## Tips
- Create events early so family members can plan travel.
- Include parking instructions and accessibility information in the description.
- If you're planning multiple events (funeral + repast), create each one separately so guests can RSVP individually.
- Use the Notes field for livestream links if offering a virtual option.

## FAQ

**Q: Can guests RSVP without a FinalWishes account?**
A: Sharing and RSVP for non-account holders is managed through the public memorial link system. Account holders can RSVP directly from the Events page.

**Q: Can I send email invitations from FinalWishes?**
A: FinalWishes can send email notifications to estate team members (executors, heirs, legal counsel) when events are created or updated. For broader invitations, copy the event link and share it through your preferred method.

**Q: What if I need to change the date or location?**
A: Click the edit button on the event and update any field. Team members with notifications enabled will be alerted to the change.
